Варианты реализации кнопок "Выбрать" в секции "Стоимость проектирования"

Вариант 1: Открытие модального окна обратной связи с указанием пакета

При нажатии на кнопку открывается существующее модальное окно обратной связи, и в сообщении автоматически указывается выбранный пакет (Базовый/Стандартный/Премиум).
// Добавить в HTML к каждой кнопке: <button class="btn pricing-btn" onclick="selectPricingPackage('Базовый', 80)">Выбрать</button> <button class="btn pricing-btn" onclick="selectPricingPackage('Стандартный', 150)">Выбрать</button> <button class="btn pricing-btn" onclick="selectPricingPackage('Премиум', 250)">Выбрать</button> // Добавить в JavaScript: function selectPricingPackage(packageName, price) { selectedPackage = packageName; selectedPrice = price; openCallbackModal('Выбор пакета: ' + packageName); // Автозаполнение текстового поля в форме setTimeout(() => { const messageField = document.querySelector('#callbackModal textarea[name="message"]'); if (messageField) { messageField.value = `Интересует пакет "${packageName}" (от ${price} руб/м²)`; } }, 100); }

Вариант 2: Прокрутка к форме заказа с подсветкой выбранного пакета

При нажатии на кнопку страница прокручивается к форме заказа, и выбранный пакет визуально выделяется (например, рамкой или фоном).
// Добавить в HTML к каждой кнопке: <button class="btn pricing-btn" data-package="Базовый" onclick="scrollToOrderForm(this)">Выбрать</button> // Добавить в JavaScript: function scrollToOrderForm(button) { const packageName = button.getAttribute('data-package'); // Сохраняем выбранный пакет sessionStorage.setItem('selectedPackage', packageName); // Прокрутка к форме (если есть секция с формой) const orderSection = document.getElementById('order-form') || document.getElementById('callbackModal'); if (orderSection) { orderSection.scrollIntoView({ behavior: 'smooth', block: 'center' }); // Открываем модальное окно, если это оно if (orderSection.id === 'callbackModal') { openCallbackModal('Выбор пакета: ' + packageName); } } // Подсветка выбранной карточки document.querySelectorAll('.pricing-card').forEach(card => { card.classList.remove('selected'); }); button.closest('.pricing-card').classList.add('selected'); } // Добавить в CSS: .pricing-card.selected { border: 2px solid #d4af37; box-shadow: 0 0 20px rgba(212, 175, 55, 0.5); transform: scale(1.02); }

Вариант 3: Показ уведомления с кнопкой перехода к заказу

При нажатии показывается красивое уведомление (toast) с информацией о выбранном пакете и кнопкой "Оформить заказ", которая открывает форму обратной связи.
// Добавить в HTML перед закрывающим тегом body: <div id="pricingToast" class="pricing-toast"> <div class="toast-content"> <i class="fas fa-check-circle"></i> <div class="toast-text"> <strong>Выбран пакет: <span id="toastPackageName"></span></strong> <p>От <span id="toastPrice"></span> руб/м²</p> </div> <button class="toast-btn" onclick="openCallbackModal('Выбор пакета')">Оформить заказ</button> <button class="toast-close" onclick="closePricingToast()">&times;</button> </div> </div> // Добавить в JavaScript: function showPricingToast(packageName, price) { document.getElementById('toastPackageName').textContent = packageName; document.getElementById('toastPrice').textContent = price; document.getElementById('pricingToast').classList.add('show'); setTimeout(() => { closePricingToast(); }, 5000); } function closePricingToast() { document.getElementById('pricingToast').classList.remove('show'); } // Добавить обработчики к кнопкам: document.querySelectorAll('.pricing-btn').forEach(btn => { btn.addEventListener('click', function() { const card = this.closest('.pricing-card'); const packageName = card.querySelector('h3').textContent; const price = card.querySelector('.price span:last-of-type').textContent.match(/\d+/)[0]; showPricingToast(packageName, price); }); }); // Добавить в CSS: .pricing-toast { position: fixed; bottom: 20px; right: 20px; background: rgba(26, 29, 41, 0.98); border: 2px solid #d4af37; border-radius: 12px; padding: 20px; box-shadow: 0 8px 32px rgba(0, 0, 0, 0.5); transform: translateX(400px); opacity: 0; transition: all 0.3s ease; z-index: 10000; max-width: 350px; } .pricing-toast.show { transform: translateX(0); opacity: 1; }

Вариант 4: Открытие боковой панели контактов с предзаполненными данными

При нажатии открывается боковая панель контактов (heroSidebar), и в форме автоматически указывается выбранный пакет.
// Добавить в HTML к каждой кнопке: <button class="btn pricing-btn" data-package="Базовый" data-price="80" onclick="selectPackageAndOpenSidebar(this)">Выбрать</button> // Добавить в JavaScript: function selectPackageAndOpenSidebar(button) { const packageName = button.getAttribute('data-package'); const price = button.getAttribute('data-price'); // Открываем боковую панель openContactSidebar(); // Предзаполняем форму после открытия панели setTimeout(() => { const messageField = document.querySelector('#heroSidebar textarea, #heroSidebar input[type="text"]'); if (messageField) { messageField.value = `Интересует пакет "${packageName}" (от ${price} руб/м²)`; messageField.focus(); } }, 300); }

Вариант 5: Комбинированный вариант (рекомендуемый)

При нажатии: подсвечивается выбранная карточка, показывается уведомление, и через 1 секунду автоматически открывается модальное окно с предзаполненной формой.
// Добавить в HTML к каждой кнопке: <button class="btn pricing-btn" data-package="Базовый" data-price="80" onclick="selectPricingPackageComplete(this)">Выбрать</button> // Добавить в JavaScript: function selectPricingPackageComplete(button) { const card = button.closest('.pricing-card'); const packageName = button.getAttribute('data-package') || card.querySelector('h3').textContent; const priceText = card.querySelector('.price').textContent; const price = priceText.match(/\d+/)[0]; // 1. Подсветка карточки document.querySelectorAll('.pricing-card').forEach(c => c.classList.remove('selected')); card.classList.add('selected'); // 2. Показ уведомления showPricingToast(packageName, price); // 3. Открытие модального окна через 1 секунду setTimeout(() => { openCallbackModal('Выбор пакета: ' + packageName); // Предзаполнение формы setTimeout(() => { const messageField = document.querySelector('#callbackModal textarea[name="message"]'); if (messageField) { messageField.value = `Интересует пакет "${packageName}" (${priceText.trim()})`; } }, 200); }, 1000); }
