# ТОПБИЛДИНГ - Официальный сайт

Сайт компании "ТОПБИЛДИНГ" - проектирование и строительство домов под ключ в Республике Беларусь.

## 🚀 Быстрый старт

### 1. Клонирование репозитория
```bash
git clone <repository-url>
cd TopBuilding
```

### 2. Настройка Telegram уведомлений

Скопируйте файл-пример и заполните своими данными:

```bash
cp config.example.js config.js
```

Откройте `config.js` и замените:
- `YOUR_BOT_TOKEN_HERE` на токен от @BotFather
- `YOUR_CHAT_ID_HERE` на ваш chat_id или ID группы

**Как получить токен и chat_id смотрите в файле `config.example.js`**

### 3. Запуск локально

Просто откройте `index.html` в браузере или используйте Live Server.
